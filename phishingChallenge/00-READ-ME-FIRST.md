## 🎮 **GAMEPLAY FLOW**

```
1. Email generates (phishing or legitimate)
2. User analyzes email (inspect URL, check sender, etc.)
3. User clicks "Phishing" or "Legitimate"
4. Score updates instantly
5. Feedback modal shows analysis
6. User clicks "Next Email"
7. Difficulty adjusts based on performance
8. Repeat!
```
