# CyberGuard / CyberAdventure

CyberGuard is a cybersecurity learning project that includes phishing-detection gameplay, progression mechanics, and a small pygame-based character module.

## Repository Structure

- `cyber-challenge/`: Main Next.js app with challenges, progression API, and leaderboard
- `phishingChallenge/`: Challenge-focused module and related components
- `app/`: Additional challenge scaffolding and examples
- `main.py`: Pygame prototype entry point

## Requirements

- Node.js 20+
- npm 10+
- Python 3.10+ (for `main.py`)

## Quick Start (Next.js app)

1. Install dependencies:
   `npm install`
2. Run the app:
   `cd cyber-challenge`
   `npm install`
   `npm run dev`
3. Open browser at:
   `http://localhost:3000`

## Python Prototype

1. Install pygame:
   `pip install pygame`
2. Run:
   `python main.py`

## Development Notes

- Keep commits focused and small.
- Do not commit `node_modules`.
- Add tests for gameplay logic when extending challenges.
